Jubilee Austen Project

< HTML 5 and CSS Website />

This project was my first real dive into creating something cool with HTML 5 and CSS. It taught me how to read a design comp using Photoshop and how to take that information and replicate it into a website.

What I loved about this project was seeing how HTML and CSS work together. They work brilliantly in tandem to produce something amazing. It was a lot of fun to experiment and see a design concept come to life in a fully functioning website.

Rouge Pickings Website

< HTML 5 and CSS Website />

My first complete solo project. Similar to the Jubilee Austen Project, I used a design comp to write up a website using HTML and CSS. This project required me to rely more heavily on the knowledge I had learned previously.

This design was full of challenges and really forced me to go beyond my current scope of knowledge and seek out answers for many of the questions I had. It was definitely a fun learning experience and I'm excited to do more of them in the future.
